# Terminal

Terminal em Python para navegar entre diretórios

## Funcionalidades

- Ver o diretório atual
- Listar arquivos e pastas
- Navegar para outras pastas (cd)
- Voltar (cd ..)
- Limpar a tela (clear)
- Sair do terminal (exit)

## Como usar

```bash
pip install -r requirements.txt
python src/main.py
```
